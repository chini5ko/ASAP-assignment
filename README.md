# ASAP-assignment

## run flask app

```
 source venv/bin/activate
 pip install -r requirements.txt
 python app.py
```

## run unit testing

```
 python -m pytest
```

## others command line

```
 source venv/bin/activate
 pip install flask gunicorn
 it push heroku main
 pip freeze > requirements.txt
 python -m pytest
```
